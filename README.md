# Polymorphism 

A beginner-friendly Java project demonstrating the concept of Polymorphism in Object-Oriented Programming (OOP) using Eclipse IDE.
This project helps in understanding how the same method can perform different tasks through Method Overloading and Method Overriding.

---

## 🚀 Features
- Demonstrates Method Overloading
- Demonstrates Method Overriding
- Runtime Polymorphism examples
- Compile-time Polymorphism examples
- Easy-to-understand Java OOP implementation

---

## 🛠️ Technologies Used
- Java
- Eclipse IDE
- Object-Oriented Programming (OOP)

---

## 🎯 Learning Purpose

This project was created for practicing OOP concepts in Java and understanding how Polymorphism improves flexibility and reusability in programming.
